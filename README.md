# ✅ Task Management Application

A React-based task management app for organizing daily activities with persistent storage.

## 🚀 Features
- Add, edit, and delete tasks
- Persistent data using localStorage
- Dynamic UI updates
- Responsive layout

## 🛠️ Tech Stack
- React
- JavaScript (ES6+)
- CSS
